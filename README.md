# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. This project showcases core frontend development skills such as DOM manipulation, event handling, and implementing game logic.

---

## 🚀 Live Demo

🔗https://vanshikajain-gt.github.io/Game-1/

---

## 📌 Features

* 🎮 Play against the computer
* ⚡ Instant game result (Win / Lose / Draw)
* 🔄 Random computer choice generation
* 🎨 Simple and clean user interface
* 📱 Responsive design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript (ES6)** – Logic & interactivity

## ⚙️ How It Works

1. User selects Rock, Paper, or Scissors
2. Computer randomly selects one option
3. Game logic compares both choices
4. Result is displayed instantly

---

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/rock-paper-scissors.git
```

2. Navigate to the folder:

```
cd rock-paper-scissors
```

3. Open `index.html` in your browser
